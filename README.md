Data and input files for the paper "Understanding halide segregation in metal halide perovskites through defect thermodynamics", [arXiv:2603.07829](https://arxiv.org/abs/2603.07829)

One file for bulk and surface calculations is provided in the input-files folder.

Relaxed structures of pristine bulk and slab, as well as the respective defective configurations, are provided in the relaxed-structures directory.
